Sobre a Oficina

Esta oficina foi desenvolvida com foco prático para introduzir conceitos essenciais de versionamento utilizando Git e GitHub, conectando o aprendizado com situações reais do mercado de trabalho.
O objetivo não é apenas ensinar comandos, mas mostrar como o versionamento se encaixa dentro do ciclo completo de desenvolvimento de software, desde a criação até a entrega em produção.

Objetivo Capacitar o aluno a:

Entender o problema que o Git resolve
Utilizar Git no dia a dia de desenvolvimento
Trabalhar em equipe utilizando GitHub
Compreender o fluxo real de desenvolvimento em projetos
Ter uma visão inicial sobre deploy e pipelines
Público-Alvo
Iniciantes com conhecimento básico de desenvolvimento
Estudantes que já tiveram contato inicial com Git
Desenvolvedores que ainda não entendem o fluxo completo de entrega
